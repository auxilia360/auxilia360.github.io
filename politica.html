1. Responsable del tratamiento

Correo electrónico: auxilia360@gmail.com
País de actividad: España

2. Datos personales que recopilamos

A través de este sitio web podemos recopilar las siguientes categorías de datos:

• Nombre y apellidos.
• Dirección de correo electrónico.
• Número de teléfono.
• Información facilitada voluntariamente mediante formularios de contacto o solicitud de información.

El usuario garantiza que los datos facilitados son veraces, exactos y actualizados.

3. Finalidad del tratamiento

Los datos personales serán tratados con las siguientes finalidades:

• Gestionar consultas, solicitudes de información y comunicaciones realizadas por los usuarios.
• Prestar los servicios ofrecidos a través de la web.
• Mejorar la experiencia de navegación y la calidad de nuestros servicios.
• Realizar análisis estadísticos sobre el uso del sitio web.
• Cumplir con las obligaciones legales aplicables.

4. Base jurídica del tratamiento

El tratamiento de los datos personales se fundamenta en:

• El consentimiento expreso otorgado por el usuario al enviar formularios o contactar con Auxilia360.
• La ejecución de medidas precontractuales solicitadas por el usuario.
• El cumplimiento de obligaciones legales.
• El interés legítimo del responsable para garantizar la seguridad y el correcto funcionamiento del sitio web.

5. Conservación de los datos

Los datos personales serán conservados durante el tiempo necesario para atender la finalidad para la que fueron recabados y, posteriormente, durante los plazos legalmente exigidos para atender posibles responsabilidades.

6. Destinatarios y cesión de datos

Con carácter general, los datos no serán cedidos a terceros salvo obligación legal.

No obstante, determinados proveedores tecnológicos pueden tener acceso a la información cuando resulte necesario para la prestación de servicios relacionados con:

• Alojamiento web.
• Servicios de correo electrónico.
• Herramientas de análisis y estadísticas.
• Servicios de seguridad y mantenimiento.

En todo caso, dichos proveedores actuarán como encargados del tratamiento y bajo las garantías exigidas por la normativa vigente.

Este sitio web puede utilizar herramientas de análisis y medición del tráfico con el fin de conocer el comportamiento de los usuarios y mejorar los servicios ofrecidos.

Estas herramientas pueden recopilar información como:

• Páginas visitadas.
• Tiempo de permanencia.

• Dispositivo empleado para acceder al sitio web.

El usuario puede configurar o rechazar determinadas cookies a través de las opciones disponibles en su navegador o mediante el sistema de gestión de consentimiento que, en su caso, se implemente en la web.
